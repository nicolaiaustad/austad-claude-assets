When doing UI or frontend work, use the `ui-ux-pro-max` skill to ensure polished, professional results that match the existing design system.

**Why:** UI changes looked ugly when done without design guidance -- mismatched visual weight, too many elements crammed together, breaking the visual flow of the page.

**How to apply:** Before writing HTML/CSS for user-facing pages, invoke the `ui-ux-pro-max` skill. Especially important for landing/onboarding pages where visual cohesion matters most. For dashboard components inside existing card layouts, matching existing patterns manually is usually sufficient.
